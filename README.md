# Machine-Learning

# 📊 Attendance Prediction using Machine Learning

## 📌 Project Overview

This project focuses on analyzing student attendance data using Machine Learning techniques. The goal is to preprocess the attendance dataset, perform data analysis, train predictive models, and evaluate their performance.

This project was developed as part of a Machine Learning milestone project.

---

## 📂 Repository Structure

```
Machine-Learning/
│
├── Attendance_Report.xls        # Dataset
├── milestone project(ML).ipynb  # Jupyter Notebook
├── README.md                    # Project Documentation
```

---

## 🎯 Objectives

- Analyze attendance data
- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Build Machine Learning models
- Compare model performance
- Predict attendance-related outcomes
